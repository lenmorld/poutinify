### Tech used

- React
- Node + Express
- Webpack, Babel
- Netlify + serverless functions

### API, Libraries used

- Yelp API
- Leaflet + OpenStreetMap (open-source alternative to Google Maps)

### Dev

Watch both UI (webpack) and server (nodemon) changes
```
$ npm install
$ npm start
```

### Build
```
$ npm run build
```
